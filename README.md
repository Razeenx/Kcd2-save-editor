# KCD2 Save Editor — PWA/html

Редактор сохранений **Kingdom Come: Deliverance II** в виде Progressive Web App.  
Работает в браузере, устанавливается на Android/iOS/Desktop как приложение.

## Как пользоваться

PWA:
1. Скачать файлы и развернуть сайт:
   [GitHub Pages](https://pages.github.com)
   [Netlify](https://netlify.com)
   [Vercel](https://vercel.com)
2. Найти файл сохранения на ПК:  
   `%LOCALAPPDATA%\Kingdome Come Deliverance II\Saved Games\`
3. Скопировать `.whs` файл на телефон (USB, облако)
4. Открыть редактор → выбрать файл
5. Изменить нужные параметры → **Сохранить файл**
6. Перенести изменённый файл обратно в папку сохранений

html:
1. Открыть файл kcd2_save_editor.html в брузере
2. Найти файл сохранения на ПК:  
   `%LOCALAPPDATA%\Kingdome Come Deliverance II\Saved Games\`
3. Скопировать `.whs` и сделать бэкап этого файла сохранения
4. Открыть редактор → выбрать файл
5. Изменить нужные параметры → **Сохранить файл**
6. Перенести изменённый файл обратно в папку сохранений
